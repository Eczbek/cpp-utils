## Example
```cpp
import std;
import xieite;

int main() {
    auto liters = xieite::units::Liter(5);
    auto gallons = static_cast<xieite::units::Gallon>(liters);

    std::println("{} L", liters.value);
    std::println("{} gal", gallons.value);
}
```
Possible output:
```
5 L
1.099845787 gal
```
