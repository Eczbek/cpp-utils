# [XIEITE](../../macros.md)\_ARCHITECTURE\_...

## Example
```cpp
#include <print>
#include "xieite/macros/architecture.hpp"

int main() {
#if XIEITE_ARCHITECTURE_VERSION_LEAST(MIPS, 4, 0, 0)
    std::println("yep");
#else
    std::println("nope");
#endif
}
```
Possible output:
```
nope
```
